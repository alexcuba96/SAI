CONFIGURACION DEL APLICATIVO

 

Se coloca la carpeta llamada GRC en la ruta donde se encuentra instalado el aplicativo normalmente es: C:\wamp\www

 

Luego se buscan los archivos que mencionan a continuaci�n y se configuran

 

Configurar la base de datos

Se modifica el archivo .env

# DB Connection

DB_CONNECTION=mysql

DB_HOST=localhost

DB_DATABASE=grc

DB_USERNAME=root

DB_PASSWORD=

DB_PREFIX=

# Environment

APP_ENV=local

APP_DEBUG=true

APP_TWIGDEBUG=true

APP_TWIGCACHE=true

APP_DISPLAYERRORDETAILS=true

Servidor activo

Protocolo smtp Puerto 25

Correr el aplicativo

Abre la consola de comandos git bash y se ejecuta el siguiente comando php –S 127.0.0.1: puerto –t public y abres el navegador con la url localhost:puerto
