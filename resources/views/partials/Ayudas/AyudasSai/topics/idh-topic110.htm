Inicio Sesion

 

El usuario ingresa al sistema con la siguiente Url http://www.gatosrugby.com.co/GRC/ 

 

la cual mostrará el inicio de sesión.

 

 

para poder ingresar al sistema ya sea como administrador o usuario general debe ingresar los datos correctamente.

 

Usuario: es el correo electrónico ingresado al momento del registro.

 

Clave: clave digitada o asignada al momento del registro.

 

Regístrate: si no posee un usuario, puede usar el link Registrarse para crear uno, una vez los administradores validen la información le habilitarán el acceso.

 

Nota: si olvido la clave revise ¿Cómo recuperar la contraseña?

 

Si los datos de inicio son incorrectos este mostrará un mensaje de validación indicando que el usuario o clave son incorrectos.

Una vez los datos estén ingresados de forma correcta este ingresará a un menú principal ya sea como administrador o usuario general.

 

Registrarte

 

Al darle click a "Regístrate" mostrará el siguiente formulario.

 

 

 

 

Nombre Usuario: nombre de la persona a la que se le está creando el usuario.

 

Documento Identidad: identificación de la persona a la que se le está creando el usuario.

 

Correo electrónico: Correo electrónico de la persona a la que se le está creando el usuario, este será el Usuario con el que debe iniciar sesión.

 

Clave: Clave de acceso.

 

Seleccione Perfil: será el perfil de acceso del usuario. Se podrá seleccionar entre miembro (Aquellas personas que hacen parte o están vinculadas a la entidad, pero no son ni jugadores ni administrativos), jugador, administrador.

 

Seleccionar Archivo: Se debe cargar la foto correspondiente al usuario.

 

Inicia Sesión: Enlace que redirecciona al formulario de inicio de sesión.

 

Una vez validados todos los campos y darle click al Registrarse el proceso estará completo y queda a la espera de que un administrador habilite el usuario.
