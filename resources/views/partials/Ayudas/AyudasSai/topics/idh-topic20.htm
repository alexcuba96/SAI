Usuarios

 

Al ingresar al módulo de gestión de usuarios aparecerá un listado con todos los usuarios registrados en el sistema la cual va a poder crear, modificar y eliminar un usuario.

 

Nota: Si es Usuario No administrador solo mostrará su propia información.

 

 

 

 

Crear

 

 

Darle click al botón Crear, aparece una ventana modal con los campos necesarios para crear un usuario con todos los campos obligatorios y sus respectivas validaciones, como lo son números, letras, campos obligatorios, email, y la foto solo permite formato jpg.

 

 

 

 

Nombre Usuario: nombre de la persona a la que se le está creando el usuario.

 

Documento Identidad: identificación de la persona a la que se le está creando el usuario.

 

Correo electrónico: Correo electrónico de la persona a la que se le está creando el usuario, este será el Usuario con el que debe iniciar sesión.

 

Clave: Clave de acceso.

 

Seleccione Perfil: será el perfil de acceso del usuario. Se podrá seleccionar entre miembro (Aquellas personas que hacen parte o están vinculadas a la entidad, pero no son ni jugadores ni administrativos), jugador, administrador.

 

Estado: Solo los perfiles administradores pueden modificar este campo, indica si el usuario está activo o inactivo.

 

Foto: Se debe cargar la foto correspondiente al usuario.

 

Search

 

este campo permite realizar una búsqueda y filtrado simultáneo y en tiempo real de los datos, por cualquier campo o palabra clave, una vez ingresada, alguna palabra clave se mostrará en pantalla un listado resultado.

 

Copy

Permite realizar una copia del listado actual mostrado por pantalla.

Excel

          Permite Exportar a Excel el listado actual mostrado por pantalla.

Csv

Permite Exportar a Csv el listado actual mostrado por pantalla.

PDf

Permite Exportar a PDF el listado actual mostrado por pantalla.

 

Editar

Editar el registro, al darle click despliega el modal donde se puede editar la información del registro seleccionado

 

 

La información que se puede editar es:

 

Nombre Usuario.

 

E-mail, bajo el riesgo del usuario, recordar que es el usuario de acceso.

 

Contraseña, bajo el riesgo del usuario, recordar que es el usuario de acceso.

 

Perfil (Solo Administradores).

 

Estado (Solo Administradores).

 

Eliminar

Eliminar el Registro, al darle permite eliminar el registro seleccionado, saldrá un mensaje de confirmación o no del proceso.
