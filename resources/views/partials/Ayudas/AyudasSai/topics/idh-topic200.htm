¿Cómo funcionan las búsquedas de registros?

 

Ingresando a cualquiera de los mantenimientos del sistema.

 

 

 

 

Ubicar el campo de búsqueda.

 

 

 

este campo permite realizar una búsqueda y filtrado simultáneo y en tiempo real de los datos, por cualquier campo o palabra clave, una vez ingresada, alguna palabra clave se mostrará en pantalla un listado resultado.

 

 

Ver filtro según búsqueda anterior  

 

          Teniendo como base la imagen anterior, se realiza una búsqueda por un apalabra clave contendida dentro del apellido del registro.

 

Si se requiere realizar otras búsquedas basta con cambiar la palabra calve y el sistema se encargará de mostrar los resultados.

 

 

 

El resultado de la búsqueda puede ser ordenado según el campo de elección con las flechas ubicadas a la derecha de cada encabezado de la columna.
