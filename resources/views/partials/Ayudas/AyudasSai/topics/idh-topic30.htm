Miembros

 

Si el usuario ha accedido a la aplicación correctamente. Ingresa al módulo de gestión de miembros, se mostrará en la pantalla un listado de todos los miembros del equipo (Jugadores, entrenadores, entre otros).

 

Nota: Si es Usuario No administrador solo mostrará su propia información.

 

 

 

Crear

 

 

Darle click al botón Crear, aparece una ventana modal con los campos necesarios para crear un usuario con todos los campos obligatorios y sus respectivas validaciones, como lo son números, letras, campos obligatorios, email.

 

 

Nombres: indica el o los Nombres del miembro que se está registrando.

 

Apellidos: indica el o los Apellidos del miembro que se está registrando.

 

Tipo Documento: indica el tipo de documento del miembro que se está registrando. Ej. cédula, Tarjeta identidad, etc.

 

Identificación: indica el número de identificación del miembro que se está registrando.

 

Teléfono: indica el número de identificación del miembro que se está registrando.

 

Dirección: indica la dirección de residencia del miembro que se está registrando.

 

Correo electrónico: indica el correo electrónico del miembro que se está registrando.

 

Fecha Nacimiento: indica la fecha de nacimiento del miembro que se está registrando.

 

Estado: indica si el miembro que se está registrando está activo o Inactivo.

 

Eps: indica la entidad prestadora de salud del miembro que se está registrando.

 

Talla Camisa: indica la talla de camisa de uniforme del miembro que se está registrando.

 

Talla Pantaloneta: indica la talla de pantaloneta del miembro que se está registrando.

 

Altura: indica la altura del miembro que se está registrando (en metros).

 

Peso: indica el peso miembro que se está registrando (En Kilogramos).

 

Tipo Sangre: Grupo sanguieneo al que pertenece el Miembro que se esta registrando.

 

Rh: Factor sanguieno (Positivo o Negativo) que posee el Miembro que se esta registrando

 

Nota: La información de Usuario, Miembro e información Deportiva (Si la tiene) estarán vinculadas.

 

Una vez completo el formulario si el usuario pulsa el botón Guardar se enviará un correo electrónico al administrador indicando que el registro se creó correctamente, posteriormente un mensaje de validación indicando que el registro se ha creado correctamente y este se verá en el listado de los miembros registrados.

 

Search

 

este campo permite realizar una búsqueda y filtrado simultáneo y en tiempo real de los datos, por cualquier campo o palabra clave, una vez ingresada, alguna palabra clave se mostrará en pantalla un listado resultado.

 

Copy

Permite realizar una copia del listado actual mostrado por pantalla.

Excel

          Permite Exportar a Excel el listado actual mostrado por pantalla.

Csv

Permite Exportar a Csv el listado actual mostrado por pantalla.

PDf

Permite Exportar a PDF el listado actual mostrado por pantalla.

 

Editar

Editar el registro, al darle click despliega el modal donde se puede editar la información del registro seleccionado.

 

 

La información que se puede editar es:

 

Nombres: Indica el o los Nombres del miembro que se está registrando.

 

Apellidos: Indica el o los Apellidos del miembro que se está registrando.

 

Tipo Doc.: indica el tipo de documento del miembro que se está registrando. Ej.cédula, Tarjeta identidad, etc.

 

Identificación: indica el número de identificación del miembro que se está registrando.

 

Teléfono: indica el número de identificación del miembro que se está registrando.

 

Dirección: indica la dirección de residencia del miembro que se está registrando.

 

Correo electrónico: indica el correo electrónico del miembro que se está registrando.

 

Fecha Nacimiento: indica la fecha de nacimiento del miembro que se está registrando.

 

Estado: indica si el miembro que se está registrando está activo o Inactivo.

 

Eps: indica la entidad prestadora de salud del miembro que se está registrando.

 

Talla Camisa: indica la talla de camisa de uniforme del miembro que se está registrando.

 

Talla Pantaloneta: indica la talla de pantaloneta del miembro que se está registrando.

 

Altura: indica la altura del miembro que se está registrando (en metros).

 

Peso: indica el peso miembro que se está registrando (En Kilogramos).

 

Una vez completo el formulario si el usuario pulsa el botón Guardar se enviará un correo electrónico al administrador indicando que el registro se creó correctamente, posteriormente un mensaje de validación indicando que el registro se ha creado correctamente y este se verá en el listado de los miembros registrados.

 

 

Eliminar

          Eliminar el Registro, al darle permite eliminar el registro selecciondo, saldrá un mensaje de confirmación o no del proceso.
